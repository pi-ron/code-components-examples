# Pricing Quote Calculator Component

When working with the Pricing Quote Calculator:

## Component Overview
A multi-input insurance premium calculator with form validation and URL parameter support.

## Key Features
- Multi-input form logic with validation
- URL parameter support for pre-filling form inputs
- Responsive design using Tailwind CSS and DaisyUI
- Real-time calculation updates

## Development Guidelines
- Use React hooks for state management
- Implement form validation with proper error messages
- Support URL parameters for default values
- Ensure responsive design across all devices
- Use TypeScript for type safety

## Props Configuration
When imported into Webflow, configure these props:
- Premium rates
- Form field labels
- Validation rules
- Default values

## File Structure
- src/components/: React components
- src/utils/: Calculation utilities
- src/types/: TypeScript type definitions
- public/: Static assets

## Testing Notes
- Test form validation scenarios
- Verify URL parameter parsing
- Check responsive behavior
- Validate calculation accuracy
